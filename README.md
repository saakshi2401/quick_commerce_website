# MERN Stack Quick Commerce Website

A full-stack quick commerce web application inspired by Blinkit, built using the MERN stack. This project provides a seamless online grocery shopping experience with user authentication, product browsing, cart management, and responsive UI design.

---

## 🚀 Features

- User Authentication & Authorization
- Product Listing and Categories
- Add to Cart Functionality
- Dynamic Cart Management
- Responsive User Interface
- Admin Product Management
- REST API Integration
- MongoDB Database Connectivity
- Secure Backend using Node.js & Express.js

---

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS / Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Other Tools
- JWT Authentication
- Git & GitHub

--- 

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/blinkit-clone.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd blinkit-clone
```

### 3️⃣ Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

---

## ▶️ Run the Application

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```

---

## 🌐 Environment Variables

Create a `.env` file in the server directory and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
